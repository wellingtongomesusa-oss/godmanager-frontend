# Master-Finance

Projeto **Next.js 14** (App Router + TypeScript + Tailwind) na pasta **`Master-Finance`**.

## Comandos

```bash
cd Master-Finance
npm install
npm run dev
```

Abra **http://localhost:3000** (porta **3000** por defeito em `npm run dev`).

## Nota

O nome da pasta usa maiúsculas (`Master-Finance`); o nome no `package.json` é `master-finance` (requisito npm).
