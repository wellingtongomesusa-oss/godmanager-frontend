# cursor-projects

## GodManager (Next.js · GodManager.One)

A app **GodManager.One**, **NEWS**, KPIs e suite imobiliária está em:

**`./GodManager/`**

```bash
cd GodManager
rm -rf .next
npm install
npm run dev
```

Abre `http://localhost:3101/manager-pro` (ou a porta que o terminal indicar — por defeito **3101**).

### Confirmação rápida

No browser: `http://localhost:PORTA/godmanager-ui.txt`  
A primeira linha deve ser: `godmanager-ui-v2-godmanager-one`

Se vês **MANAGER PRO** no menu, o `next dev` **não** está a correr desta pasta `GodManager` (ou a cache `.next` está velha).
